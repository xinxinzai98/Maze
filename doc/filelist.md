# 文件结构
在控制台使用以下命令可以生成文件结构树
```
tree -I out
```
效果如下
```
├── README.md
├── doc
│   ├── filelist.md
│   ├── info.md
│   └── property.md
├── image
│   ├── MAZE.png
│   ├── mazeIcon.png
│   └── project
│       └── MAZE.psd
└── src
    ├── DefaultParam
    │   └── menuDefault.java
    ├── Maze.java
    ├── Menu
    │   ├── menuButton.java
    │   ├── menuFrame.java
    │   ├── menuPanel.java
    │   └── menuTitle.java
    ├── mazeCore
    └── mazeEditor

```
